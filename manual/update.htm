1. Introduction

This document itemizes the procedure for making necessary changes to International Shipments web site when changing from one month to another.

Necessary changes include archiving second to last month (the two most recent months are always current on the main page) updating links, creating files and folders for new month, and updating main ("index.htm") page.

[ Top of Page ]

2. Archive old month

Create file for newly archived month (if new month is July then newly archived month will be May).

  1. Copy previous archived month (in this example, "apr2000.htm") from its folder (apr2000) into folder to be archived (may2000) then rename file "may2000.htm".

  2. OPEN "index.htm" in NoteTab Light and COPY the month of May.

    COPY everything from:

    <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=400>
    <CAPTION>MAY 2000</CAPTION>

    to:

    <BR CLEAR=left>

  3. OPEN "may2000.htm" and PASTE the month of May over the month of April in "may2000.htm"

    PASTE over everything from:

    <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=400>
    <CAPTION>April 2000</CAPTION>

    to:

    <BR CLEAR=left>

  4. With "may2000.htm" still open use "Find and Replace" to change all instances of :

    <a href="may2000/mayxx.htm">

    to:

    <a href="mayxx.htm">

    (Note: "xx" equals day of the week)

    Example:

    FIND WHAT:                 may2000/may
    REPLACE WITH:           may

3. Update links in daily files

Update links to "MAIN" in all of May's daily HTML files to point to new "may2000.htm" file (instead of to "index.htm")

  1. Open all files from MAY2000 folder into NoteTab (EXCEPT for newly created "may2000.htm").

  2. FIND AND REPLACE (in all documents):

    FIND WHAT:                 ../index
    REPLACE WITH:           may2000

  3. SAVE ALL under file menu

[ Top of Page ]

4. Create new files and folders for new month

Create folders for new month.

  1. Create folder for new month (JUL2000) within the INTLSHIP folder. 
  2. Create two new folders, within JUL2000 folder just created, called DOCS and IMAGES.

Set up month file for new month.

  1. Make a copy of "jun00.htm" from the JUN2000 folder to the JUL2000 folder. 
  2. Re-name copy "jul00.htm" 
  3. Delete June content from "jul00.htm" 
  4. Delete everything between (but not including) the first
    <br>

    and

    <center><p><a HREF="../may2000/may00.htm">Jun | </a><a HREF="../index.htm">Main Menu</a> | Jul</center>

  5. Change links at top and bottom of "jul00.htm" to point to correct month. 
    There should be two sets of the following code in "jul00.htm":

    <center><p><a HREF="../may2000/may00.htm">Jun | </a><a HREF="../index.htm">Main Menu</a> | Jul</center>

    Change the above code (in both places on "jul00.htm") to read as follows:

    <center><p><a HREF="../jun2000/jun00.htm">Jun | </a><a HREF="../index.htm">Main Menu</a> | Aug</center>

    (This will allow users to jump directly from the month of July back to June without having to go back to the "index.htm" page)

5. Update links in previous month

Links in previous month ("jun00.htm") must be updated to point to new month.

  1. There should be two sets of the following code in "jun00.htm":

    <center><p><a HREF="../may2000/may00.htm">Jun | </a><a HREF="../index.htm">Main Menu</a> | Jul</center>

    Change the above code (in both places on "jun00.htm") to read as follows"

    <center><p><a HREF="../may2000/may00.htm">Jun | </a><a HREF="../index.htm">Main Menu</a> | <a HREF="../jul2000/jul00.htm">Jul</a></center>

    (This will allow users to jump directly from the month of June to July without having to go back to the "index.htm" page)

5. Update main page ("index.htm")

Delete archived month (May) from main page and add new month (July)

  1. Convert the month of May into the month of July.

    1. Locate the following code:

      <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=400>
      <CAPTION>MAY 2000</CAPTION>

      and convert it to:

      <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=400>
      <CAPTION>JULY 2000</CAPTION>

    2. Change each day of May into the appropriate day of July, row-by-row.
      For example, following is first row of May:

      <TR BGCOLOR="#CCCCCC" ALIGN="center">
	<TD height=3D35 BGCOLOR=3D"#F0E68C"><a href=3D"may2000/may01.htm">1</a></TD>
	<TD BGCOLOR=3D"#F0E68C"><a href=3D"may2000/may02.htm">2</a></TD>
	<TD BGCOLOR=3D"#F0E68C"><a href=3D"may2000/may03.htm">3</a></TD>
	<TD BGCOLOR=3D"#F0E68C"><a href=3D"may2000/may04.htm">4</a></TD>
	<TD BGCOLOR=3D"#F0E68C"><a href=3D"may2000/may05.htm">5</a></TD>
	<TD>6</TD>
	<TD>7</TD>
</TR>

      Following is above code converted to first row of July: 

      <TR BGCOLOR="#CCCCCC" ALIGN="center">
	<TD>3</TD>
	<TD>4</TD>
	<TD>5</TD>
	<TD>6</TD>
	<TD>7</TD>
	<TD>8</TD>
	<TD>9</TD>
</TR>
      Notice all references to links and colors have been removed within all of the table data (<TD></TD>) code. Background color for each row should remain as BGCOLOR="#CCCCCC".

      Continue the above until each row of May has been converted to respective row of July.

    3. When all of the rows have been converted, locate the following code:

      <BR><BR><BR><BR><BR><BR><BR>
      <CENTER><P>Or click here to<BR><A HREF=3D"may2000/may00.htm">browse entire month of May</A></CENTER>
      <BR CLEAR="left">

      and convert it to:

      <BR><BR><BR><BR><BR><BR><BR>
      <CENTER><P>Or click here to<BR><A HREF="jul2000/jul00.htm">browse entire month of July</A></CENTER>
      <BR CLEAR="left">


  2. Move the new code for July in front of the code for June (so that, when the browser opens "index.htm", July will be the first month visible.)

    COPY (or CUT, if your brave) the following:

    <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=400>
    <CAPTION>JULY 2000</CAPTION>

    Through:

    <BR><BR><BR><BR><BR><BR><BR>
    <CENTER><P>Or click here to<BR><A HREF="jul2000/jul00.htm">browse entire month of July</A></CENTER>
    <BR CLEAR="left">

    And then PASTE in front of June:

    (Locate following code and paste where indicated)

    <HR>

    {paste here}

    <BR>
    <TABLE ALIGN=LEFT BORDER=5 CELLPADDING=5 CELLSPACING=2 WIDTH=3D400>
    <CAPTION>JUNE 2000</CAPTION>

    Note: if you used the COPY function (instead of CUT) in the above steps, be sure to go back and DELETE all of July from its original location at the end of June (or else you'll end up with two Julys on the main page.) This won't be necessary if you used the CUT function.
