Preparing Data for Final Output

After all the Header, Detail & Footer windows have been completed for a specific invoice, the data can then be sent to Word for final output. Make sure at least the Header window is open for the completed invoice, then click on the Word button and the IIG will then create an invoice, using the appropriate template, in the Document directory under the INTL_INV shared directory on the network.

The document name will be the same as the invoice number with a .DOC suffix (ie. 11672.DOC). The invoice can then be opened in Word for any additional formatting that may be necessary.
